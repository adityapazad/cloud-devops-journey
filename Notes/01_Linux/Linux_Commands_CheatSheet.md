# Linux Commands Cheat Sheet:

## 1. Navigation
- `pwd` – current directory
- `ls`, `ls -la` – list files
- `cd <path>` – change directory
- `cd ..` – go up one level
- `clear` / `Ctrl + L` – clear screen

## 2. File & Directory Operations
- `mkdir` – create directory
- `touch <file>` – create file
- `cp src dest` – copy files
- `mv src dest` – move/rename
- `rm <file>` – delete file
- `rm -r <dir>` – delete directory
- `cat <file>` – show file content

## 3. User & System Info
- `whoami` – current user
- `id` – user & group info
- `hostname` – system hostname
- `uname -a` – kernel + OS info
- `env` - list enviroment variables

## 4. System & Permissions
- `sudo <cmd>` – admin privileges

## 5. Command Utilities (Help & Docs)
- `which <cmd>` – path to executable
- `history` – previous commands
- `man <cmd>` – manual pages
- `-h` / `--help` - show command help
- `apropos <cmd>` - search manual pages

## 6. Networking & Process Monitoring
- `ip` - show/manipulate network interfaces, routing, IP info
- `netstat` - show active network connection
- `ss` - show socket stats
- `ps` - show active processes
- `who` - users currently logged into the system

## 7. Hardware & Device Information
- `lsblk` - list block devices
- `lsusb` - list USB devices
