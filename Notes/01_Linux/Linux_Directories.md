# Linux Directory Structure

- `/bin` – essential user commands (ls, cp, mv)
- `/sbin` – system admin binaries (shutdown, fdisk)
- `/boot` – bootloader + kernel files
- `/etc` – configuration files for services and system
- `/home` – home folders for users
- `/root` – root user's home directory
- `/var` – logs, caches, spool files (most logs in /var/log/)
- `/usr` – user-installed programs, libraries, documentation
- `/tmp` – temporary files (auto-cleared)
- `/dev` – device files (disks, USB drives)
- `/proc` – virtual filesystem with kernel + process info
- `/opt` – optional or third-party software
